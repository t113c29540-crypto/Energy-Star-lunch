# 啟能電子 · 中午訂餐系統

> Energy Star Electronics Co., Ltd. — 內部午餐訂餐 / 叫菜彙總 / 收款催繳工具

單一 HTML 檔、純前端、零安裝、可離線。資料只存在使用者瀏覽器的 `localStorage`,**不會上傳到任何伺服器**。

🔗 **線上版（直接用這個）**：<https://t113c29540-crypto.github.io/Energy-Star-lunch/>

---

## 功能

| 分頁 | 用途 |
|------|------|
| 🍱 今日點餐 | 選日期/餐廳、逐筆新增訂單、即時統計份數與金額 |
| 📞 彙總叫菜 | 依品項自動加總,產生可複製/列印的「叫菜稿」,打電話照念 |
| 💰 收款催繳 | 每人金額自動計算、標記已付/未付、一鍵產生 **LINE 收款清單** 與 **只列未付者的催繳訊息**(自動帶入 LINE Pay／街口／現金等付款方式) |
| 🏪 餐廳菜單 | 建立常用餐廳與菜色價格,之後每天直接挑選;內建 **Uber Eats / foodpanda** 兩大外送平台,選取時可直接點連結開啟 App 點餐 |
| 💾 備份 / 紀錄 | 匯出/匯入 JSON 備份、匯出今日訂單 CSV、查看歷史、清除資料 |

## 怎麼用

1. 直接開 `index.html`（雙擊即可,或用線上版網址）。
2. 「今日點餐」選餐廳 → 大家把餐點加進去。
3. 「彙總叫菜」複製叫菜稿,打電話給餐廳。
4. 餐到了,到「收款催繳」標記誰付了錢,沒付的一鍵催繳。

## 重要提醒

- **資料存在本機瀏覽器**：換電腦、清快取、用無痕視窗 → 資料會不見。請定期到「備份 / 紀錄」**匯出 JSON** 保存。
- **單機版定位**：適合「一位訂餐負責人用一台裝置收單」。若要每位同事各自用手機即時下單合併,需另做雲端版。
- 換裝置時:在舊裝置「匯出全部資料」,在新裝置「匯入」即可接續。
